# Projeto skeleton rails 7


##Pré requisitos

 > Ruby 3.0.0

 > Rails 7.0.4.2

##Instalação

 - Faça um clone do Projeto
 - Configure o arquivo do banco de dados
   ``config/database.yml```
 - Execute em sequência no terminal os comandos:
  ``` bundle
      rake db:create db:migrate
      rails s
  ```

  Então acesse via navegador
    ```localhost:3000```
